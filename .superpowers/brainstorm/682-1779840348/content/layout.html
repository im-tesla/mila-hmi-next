Key Design Decisions

Always-visible quick route chips

  • Pinned at bottom of map, horizontally scrollable
  • One-tap to see nearby POIs on map — no search required
  • 6 categories: Gas, Food, Parking, EV, Grocery, Rest/Hotel
  • Polish query strings for better local results

Search bar as top pill

  • Collapsed: slim glass pill with "Where to?"
  • Focused: expands with backdrop blur + autocomplete dropdown
  • Instant results (no debounce lag — abort previous request instead)
  • Typing starts geocoding immediately, cancels stale requests

What goes away vs. what stays

Keeps & improves

  • Mapbox GL map (working great)
  • Route polyline + destination marker
  • Lane guidance strip (repositioned, fixed)
  • Route panel with ETA + turn-by-turn

Removes / replaces

  • QuickRoutes embedded inside search dropdown → standalone bottom strip
  • useReducer state machine → simpler useState + callbacks
  • v6 POI geocoding with strict filter → v5 with types=poi
  • driving profile → driving-traffic
  • All console.log debug spam
  • Dead Settings → Navigation tab (until wired later)
